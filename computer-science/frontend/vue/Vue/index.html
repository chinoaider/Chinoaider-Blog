# 基础问题

得到项目一开始大部分运行失败的情况，一般都是依赖问题，

用 **【git】才能直接使用【rm】命令清除本地依赖缓存，或者删除项目目录下的【node_modules】** 文件夹

重新安装依赖

# Missing script: "dev"

# 原因

Script 脚本没有 **【dev】命令而报错，检查【script】** 列表然后再运行 npm run 命令

# 解决方法

打开 **【package.json】文件，找到【scripts】** 列表，将其中的 serve 改为 dev ，或者直接改用 npm run serve 命令启动，其实就是一个自定义启动脚本命令，无非就是指令不一样，实际效果一样

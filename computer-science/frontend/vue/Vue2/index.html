# 插值语法

data 中的 name 属性的值通过双花括号语法 {{name}} 与 DOM 绑定，当数据变化时视图自动更新

{{name}} 双花括号中要写的是 js 表达式，也可以读取 data 中的所有属性，常见场景：文本插值、简单逻辑处理

# 指令语法

# v-bind:

v-bind: 可简写成 : ，作用是动态绑定 HTML 属性或组件属性到 Vue 实例的数据上‌

也可用于动态绑定 class 和 style 等标签属性

  • ‌ :class 根据条件切换类名，采用对象语法 {active:isActive} ，当 isActive 为 true 时添加 active 类
  • {} 不能独立存在，必须作为指令的参数值使用（ v-bind: ），常见场景：计算属性，方法返回对象或数组

# 准备工作

# 安装插件

sd-webui-controlenet：

ControlNet 是一种 “辅助式” 的神经网络模型结构，通过在 Stable Diffusion 模型中添加辅助模块，从而引入 “额外条件” 来控制 AI 绘画的生成过程

GitHub 地址：https://github.com/lllyasviel/ControlNet

adetailer：

ADetailer 是稳定扩散 webui 的扩展，可执行自动遮罩和修复。它类似于 Detection Detailer。

GitHub 地址：https://github.com/Bing-su/adetailer

sd-webui-depth-Lib ：

Depth 深度贴图库和手部姿势库

GitHub 地址：https://github.com/jexom/sd-webui-depth-lib

建议直接在安装新扩展里搜索安装

# 配置 ControleNet

想要正常运行 ControleNet 插件需要先把模型下载后放到此路径下 【.\sd-webui-aki-v4.4\models\ControlNet 】

秋葉整合包： https://pan.quark.cn/s/2c832199b09b

想正常使用 openpose 还需要下载这几个模型文件放进以下路径【.\sd-webui-aki-v4.4\extensions\sd-webui-controlnet\annotator\downloads\openpose】

直接复制名字在【https://huggingface.co/ 】 里搜索就找得到

facenet.pth

body_pose_model.pth

control_sd15_openpose.pth

hand_pose_model.pth

# 添加手部姿势

后面需要用到，添加完需要重启才生效所以先安装

如果觉得手势太少或者找不到需要的姿势，可以在该网站找 【https://civitai.com】直接搜【hand】就有了

900 Hands Library for Depth Library | ControlNet - v1.0 | Stable Diffusion Other | Civitai

下载后将文件丢进以下目录即可，重启 webui【.\sd-webui-aki-v4.4\extensions\sd-webui-depth-lib\maps】

# 姿势、手部重绘

# ControlNet

方法 1：

  1. 先随便生成一张手部错误结构的图片

  1. 启用 ControlNet，把刚刚生成的图片拖进去，

控制类型： 选择 OpenPose ；

预处理器：openpose_face ；

模型：sd15_openpose；

点击中间的 💥 图标，生成结构预览，点击旁边的下载按钮把图片保存下来

到这一步，可以用这个方法生成固定姿势的图片

  1. 接着来到 Depth Library，选择一个手部姿势添加、调整大小、位置、角度，保存为 png 格式

  1. 把它放进 ControlNet 单元 1 里，启用 ControlNet，控制类型：Canny；预处理 Canny；模型：sd15_Canny，单击中间的 💥 图标，生成结构预览

  1. 把随机数种子固定一下，再点生成就能重绘出一张固定手势和姿势的图了，如果生成的效果不太满意可以调整权重参数

# Adetailer

方法 2：

使用 Adetailer 修复，无论是脸部表情、手部、人物都可以使用 Adetailer 进行重绘修复，以下是对照表

型目标mAP 50mAP 50-95

face_yolov8s.pt2D / 逼真的面部表情0.7130.404
hand_yolov8n.pt2D / 写实的手0.7670.505
person_yolov8n-seg.pt2D / 写实的人物0.782 （bbox） 0.761 （掩码）0.555 （bbox） 0.460 （蒙版）
person_yolov8s-seg.pt2D / 写实的人物0.824 （bbox） 0.809 （蒙版）0.605 （bbox） 0.508 （掩码）
mediapipe_face_full逼真的面部表情--
mediapipe_face_short逼真的面部表情--
mediapipe_face_mesh逼真的面部表情--
  1. 勾选 Adetailer，模型选择 hand_yolov8n.pt，调整下面的重绘参数来达到你需要的效果，

重绘蒙版边缘模糊度：10

局部重绘幅度：0.45

固定随机数种子，添加提示词

  1. 可以配合高分辨率修复来达到效果

  1. 开始生成，生成过程中会自动识别手部进行重绘修复

  1. 如果还是达不到预期效果，可以用图生图局部重绘，用画笔涂出要重绘的部位，注意预留具生成后的大小，取消固定随机数种子

  1. 用 Depth Library 的添加手部预设，调整位置、大小、角度

  1. 和前面 ControlNet 的步骤一样，控制权重：1.45；控制模式：更偏向 ControlNet；开始绘制

# 原项目

# Text-generation-webui

前往 GitHub 仓库下载： Text-generation-webui 原始项目：https://github.com/oobabooga/text-generation-webui

用于大型语言模型的 Gradio Web UI。

根据自身硬件需求下载对应版本

NVIDIA 显卡：

  • 新型号 GPU 建议使用 CUDA 12.415
  • 旧型号 GPU 及驱动版本较低的系统建议使用 CUDA 11.712

AMD/Intel 显卡：

  • 推荐使用 Vulkan 版本构建 3

仅使用 CPU：

  • 请选择 CPU 专用构建版本 3

根据需求下载对应版本，我是 NVIDIA 显卡，环境是 Win10 这，里我选择 windows-cuda11.7 版本

# 配置参数

我的电脑配置

显卡：RTX2060

显存：6g

CPU：AMD 74800h

内存：16g

以下是我的参数配置

Model ：sakura-7b-qwen2.5-v1.0-iq4xs.gguf 、qwen2-7b-instruct-q4_k_m.gguf

Model loader ：liama.cpp

gpu-layers ：拉满

ctx-size ：2048

cache-type ：q4

flash-attn ：勾选

参数根据电脑配置而定，懒人包里有具体对应的配置参数文档

看链接: https://pan.baidu.com/s/1GISOOn7I4CcHgSCxKvUj5w?pwd=scnx 提取码: scnx

配置完后点击 Load 在后端窗口中可以看到加载配置成功

# LunaTranslator

LunaTranslator 的 github 仓库：https://github.com/HIllya51/LunaTranslator

# HOOK

  • HOOK 支持使用 HOOK 方式获取文本。对于部分游戏引擎，还支持内嵌翻译。还支持提取部分模拟器上运行的游戏的文本。对于不支持或支持不好的游戏，请提交输入

默认 HOOK 方式，选择游戏

翻译设置里模型选择：Sakura 大模型

改成你的本地 Text-generation-webui 项目端口：默认 5000 无需改

选择游戏后进入游戏勾选需要翻译的对应文本

如果游戏支持内嵌翻译选项也可使用内嵌翻译，内嵌翻译出现乱码字符，修改游戏字体即可解决

也可用于翻译生肉番，使用 player 播放器，使用 LunaTranslator 翻译字幕

推荐几个用下载生肉番的网站，可用于学习英语和日语

下载生肉番网站： https://nyaa.si/

下载生肉番字幕： https://kitsunekko.net/

查寻与管理番剧收视进度： https://bgm.tv/anime

# OCR

  • OCR 支持 ** 离线 OCR 和在线 OCR**

勾选 OCR 和本地 OCR

如果需要翻译英文，切换英文语言模型

圈选字幕范围，识别翻译

可隐藏范围框

# 计算机积极拒绝 10061

一般是代理问题，看看代理有没有开着，不然就是代理软件问题

我这里是 clash 的老 bug 了，因为没更新版本导致，如果重启之前没有关闭或退出 clash 会导致断网，需要启动 clash 才会恢复网络

# 解决方法

把代理切换到 direct 即可，或者把 clash 版本升到最新版本（推荐升级版本）

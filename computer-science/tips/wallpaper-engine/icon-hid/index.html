# 前言

你是否有这样的烦恼？

亲戚家小孩一来就翻你的电脑玩不小心看到了某些不美好的画面 （你老婆真棒！）

被同事看到什么不正经的软件正在后台运行（这粉色的 APP 是什么）

老板一走开你就笑眯眯，一过来就是在看桌面（来不及切壁纸的屑）

如果没有，请直接划走不用看了～

如果有，这些都不是问题～

接下来我将教你怎么隐藏系统托盘图标来掩盖你的罪行

下面，以 wallpaper engine 为例

  1. 首先打开注册表，win + r 键输入 regedit ，也可以 win + q 打开菜单输入 regedit，注册表会出现在第一个直接打开

  2. 在注册表顶部路径栏输入路径： HKEY_CURRENT_USER\SOFTWARE\WallpaperEngine ，按下回车，焦点会跳到对应路径目录下

  3. 右键 WallpaperEngine 👉 新建 👉 DWORD (32 位) 值

  1. 将新建的键值命名为：hideTrayIcon （注意不能有空格）

  1. 双击这个 hideTrayIcon ，把值修改为 1 （如果不想隐藏了，把值改为 0 就行）

  2. 以上步骤完成后，重启 Wallpaper Engine ，系统托盘图标就隐藏了（如果要隐藏其他软件的托盘图标也是同理）

  1. wallpaper engine 可以设置打开窗口的热键，壁纸也可以设置热键，配合这个隐藏托盘图标，有一个小缺点，就是运行中的程序任务管理器可以找到，虽然没办法做到天衣无缝，但是足够你用了，正常人也不会进到你的任务管理器里找东西，或者你有办法锁住任务管理器，也是一个解决的方法

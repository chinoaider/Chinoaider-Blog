# 前言

当你在 Wallpaper Engine 上看到一张喜欢的壁纸，想要提取这个壁纸的素材图片来收藏和使用，或是想去编辑这个壁纸，让这个壁纸更好看更加符合自己心里的要求，这时你就需要使用 RePKG 来提取 PKG 文件，将 PKG 转换为壁纸引擎项目，把 TEX 转换为 image，转储 PKG/TEX 信息

所需文件：Releases · notscuffed/repkg (github.com)

# 将 PKG 转换为壁纸引擎项目

首先需要去 GitHub 下载 PKG 解包器 / TEX 转换器

下载解压后，你就会得到一个 RePKG.exe 文件，新建一个文件夹并命名为 RePKG（文件夹名称随意） ，然后把 RePKG.exe 丢到里面

选择一张壁纸，右键 在资源管理器中打开

同样需要把壁纸文件复制粘贴到这个文件目录下

接着按住 shift + 右键 选择 在此处打开 Powershell 窗口，输入以下命令，回车执行

那么你就会得到一个 output 文件夹，里面就是你需要的壁纸引擎项目文件

# 整合包教程

记录一下 GPT-SoVITS AI 音色克隆的使用过程，GPT-SoVITS 是花儿不哭大佬研发的低成本 AI 音色克隆软件。目前只有 TTS（文字转语音）功能，将来会更新变声功能。

此笔记仅为个人学习记录使用

这里只简单记录一下使用流程和需要注意的点，详细请参考：

教程文档 ：www.yuque.com/baicaigongchang1145haoyuangong/ib3g1e

GPT-SoVITS 开源 github 发布地址：https://github.com/RVC-Boss/GPT-SoVITS

云端训练地址 ：https://www.codewithgpu.com/i/RVC-Boss/GPT-SoVITS/GPT-SoVITS-Official

# 准备工作

训练推理整合包地址：

推荐直接下载 GPT-SoVITSv3lora ，你想用 v1、v2 还是 v3 版本，通通都有

百度云盘：https://pan.baidu.com/s/1OE5qL0KreO-ASHwm6Zl9gA?pwd=mqpi

UC 浏览器：https://drive.uc.cn/s/a1fd91ae0a4f4

素材需求：

  • 尽量找清晰、干净、没有背景音的干声

  • 慎用三次元明星声音素材，仅学习使用

# 处理原音频

使用 UVR5 处理原音频（如果原音频足够干净可以跳过这步，比如游戏中提取的干声）

下载完整后包解压运行目录下 go-webui.bat 批处理文件，等待打开浏览器 ui 界面，如果打不开就是缺少依赖，去更新依赖具体看官方教程

# 语音切分

  1. 将音频素材路径填入框中，一般默认参数即可，如果切出来的音频太长或太短，可以修改最短切割间隔 min_interval 这个值，以毫秒为单位，切出来的素材 3~10 秒就可以使用了，太短或太长也没事，后面可以修改

# 人声分离

  1. 直接点击开启人声分离

上面已经标注的很清楚了，主保人声选择 HP5，这里直接选 HP5，输入待处理音频文件夹路径，就是刚刚语音分切输出的文件夹路径，指定一下输出人生文件夹，不然两种音频会混在一起，选择 wav 格式，点击转换

转换完毕，这时可以前往人声输出的文件夹下进行音频筛选、处理，推荐用 au 处理多段音频更方便，先把需要的人物音频挑选放到一个文件夹里，其余的角色音频去留取决你自己，日后可能需要训练哪个角色也说不定可以先留着

# 语音降噪

  1. 语音降噪其实不太推荐，使用后总感觉声音就变了个人似的，看情况，如果音源噪音过大，且找不到合适音源的情况可以用一下

# 语言识别

  1. 将处理好的音频文件路径输入到框内，点击开启语言识别，在./output/asr_opt 文件夹下会生成一个带音频路径和音频字幕的文件

# 音频标注

  1. 复制刚刚生成出来的 list 文件路径粘贴到框内，就在./output/asr_opt 下，如何点击打开音频标注 webbui

界面是英语，可以用翻译插件翻译一下，这里可以简单的音频合并、分割、删除处理，合并音频就是勾选两个以上的音频然后点击合并音频即可，还有有一些段落标注错误的、语言识别错误的可以修改一下，修改过后记得保存文件和提交文本建议做一保存一下，因为是页面关系刷新一下就全变回去了，下一个索引意思就是还有下一页，别忘修改后面的

# 分割音频

一般用来简单分割音频一段音频内有多个角色声线的，把不要的声线分割出来然后删除，或者音频过长也需要切割一下一般 10 秒以上的，选择一个音频然后看要从哪里切割，修改音频切割点的值，看切割点大概是在几秒，这个 audio Split Point 就填多少

修改完后记得关闭，就是再点一下关闭这些功能的 webui 后台，有一些一直开着会占用显存

# GPT-SoVITS TTS

切换到 GPT-SoVITS TTS 选项卡，填写实验名，选中版本，填写标注文件和训练集音频文件目录路径，点击开启训练集格式化一键三联，等待完成

选中微调训练选项卡，这里就要看你的配置来填写了，sovits 训练建议 batch_size 设置为显存的一半以下，高了会爆显存。

比如我 6g 显存 sovits 训练就填 3，GPT 训练就填 1，sovits 训练和 GPT 训练不能一起开启，除非你有两张显卡

首先设置 batch_size，sovits 训练建议 batch_size 设置为显存的一半以下，高了会爆显存。**bs 并不是越高越快！**batch_size 也需要根据数据集大小调整，也并不是严格按照显存数一半来设置，比如 6g 显存需要设置为 1。如果爆显存就调低。当显卡 3D 占用 100% 的时候就是 bs 太高了，使用到了共享显存，速度会慢好几倍。

dpo 大幅提升了模型的效果，几乎不会吞字和复读，能够推理的字数也翻了几倍，但同时训练时显存占用多了 2 倍多，训练速度慢了 4 倍，12g 以下显卡无法训练。数据集质量要求也高了很多。如果数据集有杂音，有混响，音质差，不校对标注，那么会有负面效果。

# TTS 推理

开启 TTS 推理 webui

选择刚刚训练出来的模型，找不到就刷新一下，上传一段素材音频和填写那段音频的文本到参考文本框里

复制一段话上去，语速和停顿可以自己调整

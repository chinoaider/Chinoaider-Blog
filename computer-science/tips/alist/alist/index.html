# 安装配置

打开下载的文件所在位置，点击所在文件夹上方的导航栏，输入 cmd 敲回车

输入 alist.exe server 启动 alist 服务 ，弹出防火墙点确定，就打开 alist 了

在浏览器输入 127.0.0.1.5244 访问 alist 界面

# 添加存储

推荐几个常用好用的网盘：阿里云盘、天翼云盘

# 添加阿里云盘

点击存储空间选择阿里云盘

只需填挂载路径和 token 令牌就行了

挂载路径： /阿里云盘

前往获取刷新令牌：https://alist.nn.ci/tool/aliyundrive/request

也可以只挂载某个文件夹，只需要获取文件夹的 id 填到 根文件夹 ID 就行了

打开阿里云盘官网，点击进入要设置的文件夹时点击 URL 后面的字符串填到根文件夹 ID 就行了

# 添加天翼云盘

添加天翼云盘的方法也是差不多，只需要填天翼登陆的用户名和密码就行了 ，选择个人云然后添加

添加其他网盘都差不多，详细查看 alist 文档 就有教程，直接看就行了

# RaiDrive 工具

接下来演示把 Alist 网盘用 Webdav 方式映射到电脑，实现本地读写、扩容

这里使用 RaiDrive 软件来挂载 WebDav，界面简单易懂，不需要翻墙就能下载

前往下载： https://www.raidrive.com/

也有其他的软件可以挂载 WebDav 详细请查看 Alist 文档

下载安装后点击 Add 👉 NAS 👉 WebDAV

  • Drive：选择盘符和命名

  • Address：取消选择，填入你的 alist 后台的地址和端口号

  • path：填 /dav

  • Account：alist 的账户和密码，点击 ok 即可

这是我添加好的配置

连接成功后，就可以看到电脑多了一个盘，就可以直接对文件进行操作，把文件拖进去上传或把文件复制下载出来

最新版本的 RaiDrive 工具可能会带有广告，可以下载旧版本没有广告

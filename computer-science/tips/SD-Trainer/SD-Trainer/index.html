# LoRA 模型训练

记录一下大模型训练一键包 SD-Trainer 的使用过程笔记

秋葉大佬这次带来了大模型训练一键包 SD-Trainer ，非常推荐，训练 LoRA 超级方便快捷，不需要配置环境解压即用

此笔记仅为个人学习记录使用

参考：B 站秋葉 aaaki： https://www.bilibili.com/video/BV1iM4y1y7oA/

# 准备工作

  1. 下载 SD-Trainer

专业的 LoRA 模型 / 大模型训练一键包 SD-Trainer

夸克盘（最新）：https://pan.quark.cn/s/b4ccd1f635b6

百度盘（更新慢）：https://pan.baidu.com/s/1TBaoLkdJVjk_gPpqbUzZFw 提取码：p8uy

gutub ：https://github.com/Akegarasu/lora-scripts

  1. 规划方向

先规划想训练什么样的模型，，或是某种场景、某种艺术风格、色彩风格等都需要按照其特点进行调度。

训练人物

训练某个角色 、物体、姿势、服装等模型，那图片素材并不是越多越好，一般建议 20 张左右即可

素材要求：不同角度，不同背景，不同姿势，不同服饰，清晰无遮挡的图片，背景越简单越好，不要有水印，甚至可以个抠图，这样训练出来的模型效果会更好

训练画风

训练某种风格画风，某种艺术风格、色彩风格等，具有某种特征的模型，一般建议 50 张左右即可

素材要求：同种样式、样貌、特征非常接近的素材占比不要过高，不然很容易会造成权重偏移

  1. 素材收集

我这里准备了一组角色图片，你可以准备自己喜欢的角色的图片，也可以找个视频去截图或者直接在网上找角色图各个画师找同人图等

素材收集通用要求：

  • 尽量找无遮挡、无水印的图片作为训练素材，注意人物面部要清晰，避免头发、手等遮挡
  • 尽量找画质清晰、分辨率高的图片
  • 如果商用，注意素材版权问题和慎用肖像

# 批量图片裁剪工具

可以使用修改图片工具对图片尺寸进行批量处理，至少保证一边为 512px，但必须是 64 的倍数。横竖图可放一起训练

在线批量图片裁剪网站： https://www.birme.net/

可以手动调整裁剪框位置更贴合预期效果，也可根据需求设定图片宽高，我这里 512x768 正好是 64 的倍数，如果显存较小的可以调小分辨率，zip 格式保存下载下来即可

# 运行 SD-Trainer

下载【准备工作】解压后先强制更新再启动脚本，如果没 VPN 可以用下面的国内加速路线更新

更新完点击打开启动脚本会自动打开浏览器，模型训练器 ui 界面

# 文件规范及概念

将准备的图片放到 【./train/Aris/8_Aris】 这样一个路径下，需要放要在 train 这个文件夹里，没有就自己在 lora-scripts 下创建 train 文件夹

格式：./train/Aris（画师名、角色名、概念名随便起也行）/8_Aris（数字_名称）

重复次数 Repeat ：每一张素材的重复次数需要在文件夹名以【 数字_名称 】的方式命名，如训练集 Aris 重复次数 8 ，则文件夹命名为 8_Aris

训练轮数 Epoch ：比如我有 21 张图片，整个训练集按照重复次数训练一张素材就是 8 次，8 轮就是 21*8 等于 168，就是说 AI 一共需要重复训练 168 次

# SD 自动打标

接着将图片存放路径复制粘贴到 path 里，其他参数不用改，点启动给图片生成标签

这时你会发现文件夹里每张图片后面会多出一个文本，这个文本就是打标器生成的图片标签词

# 标签编辑器

这些生成的标签文本，里面有一些可能是我们不需要的标签，一个一个的去改太麻烦了，推荐一个标签编辑工具

BooruDatasetTagManager 是一个简单的标签编辑器，用于创建和编辑用于训练超网络、嵌入、LoRA 等的数据集。用户可以从零开始创建数据集，仅使用图像，或者使用程序编辑通过自动标签（如 wd14-tagger、stable-diffusion-webui 等）创建的数据集。该编辑器主要针对 booru 风格的图像数据集。

前往 github 下载：https://github.com/starik222/BooruDatasetTagManager.git

像这种乱打的标点符号和一些不需要的词就可以点旁边的❌直接删掉，这样能保证标签词的清晰和准确性

也可以添加标签，比如加一个打标词（收敛词）进去，方便后面调用也可以不加，看个人需求效果只是用于方便调用 Lora，这个标签不能是已有的词，然后 CTRL+S 保存退出即可

# 开始训练

刚开始用推荐先用新手模式后面理解深了再用专业模式优化细节，后面再出一期专门讲专业模式的

pretrained_model_name_or_path ：

这个底模就是你的 Lora 模型会套用你使用的 Stable Diffusion 模型，基于你用的底模来训练出你的 Lora 模型

train_data_dir ：

改成你的素材路径，输入第一层文件夹路径即可

resolution ：

图片分辨率，是多少就填多少

save_every_n_epochs ：

一般填 1~2，每两轮就会保存一次模型，到时会保存出很多模型，可以根据情况设置

max_train_epochs ：

一般训练人物是大于 10，训练画风大于 15，具体看情况我这里 21 张图我填 12，前面算出 168 步再乘于 12 就是 2016 次

enable_preview ：

关闭训练预览图，开启会消耗显存资源会拖慢你的训练时间

其他都不是新手接触到的，默认即可，设置完可以将参数保存成预设，方便下次使用

# 训练完成

从图中可以看出和我们预算的一样，AI 需要训练 2016 步，训练完成前往 output 查看，这些就是训练出来的 Lora 模型

接着就可以去 Stable Diffusion WebUI 试试哪个模型的效果比较好，哪个效果好就用哪个，过程就不赘述了【sd-webui-aki Ai 绘图】

打一个尖括号【<】会出现 Lora 模型选项供你选择，选择【Aris-000010】调用我们训练的 Lora 模型，【ArisLora】是我设置的收敛词出来的效果会更好，再稍细化一下提示词

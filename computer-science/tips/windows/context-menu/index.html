# 前言

此笔记仅为个人学习记录使用

可以通过这个方法定制右键菜单，只需将值修改成所需目标程序安装路径即可

如何管理右键菜单，直接下载一个右键菜单管家，或者通过注册表修改删除（注册表谨慎操作，操作前先导出备份），推荐前者

# 配置右键打开文件

  1. Win+R 打开运行，输入 regedit ，打开 注册表 ，复制 HKEY_CLASSES_ROOT\*\shell 到搜索栏粘贴回车或手动定位到 shell 分支，如果没有 shell 分支，则在 * 下点击右键，选择 “ 新建 － 项 ”，建立 shell 分支。
  2. 右键 shell 新建项 VisualCode ，在右侧窗口的 “默认” 双击，在数据里输入 “ 用VSCode打开 ”。
  3. 在 “ VisualCode ” 下再新建 Command 项，在右侧窗口的 “默认” 键值栏内输入程序所在的安装路径，例如： "程序所在的安装路径\Microsoft VS Code\Code.exe""%1" ，你们可以右键 vscode 查看属性找到路径（其中的 %1 表示要打开的文件参数）
  4. 配置图标，在 VisualCode 项上新建 可扩充字符串值 ，命名为 Icon ，双击，把 "程序所在的安装路径\Microsoft VS Code\Code.exe" 添加数据就可以了。

# 配置右键空白处打开文件夹

  1. Win+R 打开运行，输入 regedit ，打开 注册表 ，复制 HKEY_CLASSES_ROOT\Directory\Background\shell\ 到搜索栏粘贴回车或手动定位到 shell 分支，如果没有 shell 分支，则在 * 下点击右键，选择 “ 新建 － 项 ”，建立 shell 分支。
  2. 右键 shell 新建项 VisualCode ，在右侧窗口的 “默认” 双击，在数据里输入 “ 用VSCode打开文件夹 ”。
  3. 在 “ VisualCode ” 下再新建 Command 项，在右侧窗口的 “默认” 键值栏内输入程序所在的安装路径，例如： "程序所在的安装路径\Microsoft VS Code\Code.exe""%V" ，你们可以右键 vscode 查看属性找到路径（其中的 %1 表示要打开的文件参数）
  4. 配置图标，在 VisualCode 项上新建 可扩充字符串值 ，命名为 Icon ，双击，把 "程序所在的安装路径\Microsoft VS Code\Code.exe" 添加数据就可以了。

用 moonlight 将手机与电脑串联起来局域网投屏，再用 zerotier 进行内网穿透实现异地组网

# 准备工作

下载显卡配套软件

GeForceExperience 官方：https://www.nvidia.cn/geforce/geforce-experience/

前往 GeForceExperience 官方下载显卡配套软件，打开、在设置中可以看到 SHIELD 选项卡（如果没看到这个选项可能是版本问题）

在 SHIELD 选项卡下点击【添加】，创建一个电脑桌面作为串联对象，

  • 目标：‪C:\Windows\explorer.exe（设置为资源管理器）

  • 起始位置：C:\Windows\

前往下载 app

moonlight 官方：https://github.com/moonlight-stream/moonlight-android/releases

前往 moonlight 官方下载 app，在手机上打开 app，点击【+】图标，手动添加电脑：填写电脑 IP 地址，之后会弹出配对窗口

在 PC 端填写 PIN 码进行配对，即可

# 内网穿透

zerotier 官方：https://www.zerotier.com/download/

下载安装 exe 程序，登录 zerotier，创建网络，会得到一个 Network ID，这个用于加入网段，手机在 app 上也是这个添加方式

加入成功后会出以下列表，打上勾，备注好设备，做完可能需要重新打开一次服务，不然可能会串联失败

之前内网穿透串联失败我怀疑是网段问题，换成 10.147... 网段就成功了，后面再验证一下是不是这个问题

然后在 moonlight 填上 IP 地址就可以进行串联了，IP 地址可以在任务管理器里查看，方便快捷

还有另外一种方式： Taliscale 群晖内网穿透

# 前言

家里有台旧主机，显卡坏了，

用起来很卡，丢了又可惜

思来想去拿来当 nas 主机用最合理，不需要太高的性能。

刷了个黑群晖，用起来还是不错的，

能做内网穿透，在外面也能访问 nas 的硬盘，

适合出差办公，也方便使用公司电脑办公，可用作文件传输存储分享等各种用途

接下进入教程

# 准备工作

# 硬件需求

需要一台 nas 小主机，可以是旧的电脑主机也可以自己 diy 一个，推荐蜗牛星际，适合新手入手学习使用，部分品牌机电脑或服务器可能会出现各种定制主板的不兼容毛病，建议采用正规 DIY 渠道配件。尽量不购买品牌机安装黑群晖（部分品牌会锁 BIOS 安装不了其它系统，或者设置非常麻烦），不需要无线网卡，网线直连，这样网络比较稳定

需要一个空 U 盘，这个 U 盘是作为黑群晖系统引导启动盘来使用，每次打开 nas 主机都会通过这个 U 盘来引导进入群晖系统，所以必须一直插 nas 主机在上面

需要至少一块机械硬盘，最好是两块，群晖双盘位 nas 安装系统时默认给你开了 shr 或者 Raid1，两块盘互为备份。不想要的话可以在系统装好后进入 “储存空间管理员” 删除储存空间重建 basic（无数据保护）储存池。一些比较重要数据比如照片什么的也可以备份在云盘，后面再说怎么把云盘当作备份盘来使用

# 兼容问题

自己 DIY 渠道配件 nas 主机的话可以问客服是什么型号是否兼容等问题，如果是自己的旧主机，则需要了解你的主板和 cpu 的型号是否兼容，请自行百度相关参数

举个例子，我的主机配置：

cpu：i5-3450

主板：爱国嘉 b75

内存：两条 8G

硬盘：西数 1T

ps: 在技嘉辱华前也没有觉得技嘉的东西有多好用，没办法很久以前买的了那时候还不是很懂被实体店坑的，在这里抵制爱国嘉所有产品，但真金白银买的东西不用又浪费，当作废物利用吧

爱国嘉 GA-B75M-D3V 芯片组或 北桥芯片 Intel B75 CPU 插槽 LGA 1155 支持 CPU 类型 支持 Core 三代 i7/i5/i3,Core 二代 i7/i5/i3 系列处理器，物理群晖不支持 9 代以上爱国嘉和微星主板，这里 B75 是支持的

CPU i5-3450 是 3 代酷睿，一般 4 代及以后处理器可选 918、920 引导，3 代及以前处理器可选 3615、3617 引导，或者都可试试，cpu 参数自行百度，还有一点，如果考虑功耗的话，推荐 J N D 开头的工控机 CUP（只有零售桌面型号，J N D 开头的工控机 CUP 功耗才低）

如果不知道自己 cpu 的年代可以点此连接查看：https://www.mydrivers.com/zhuanti/tianti/cpu/

RAID 控制卡是一种 磁盘阵列卡 ，它的核心就是 RAID 控制芯片 。 随着 RAID 技术的发展，现在的 RAID 控制卡不再局限提供 SCSI 一种磁盘接口，在 PC 机中常用的 IDE 和 SATA 接口 现在也可全面支持 RAID 技术了，而且在中低档 磁盘阵列 中应用非常广，特别是新兴的 SATA 接口的 RAID 控制卡。

群晖只支持软 RAID（并非主板的 RAID）和部分有驱动的硬件 RAID 卡，但是非常不建议硬件 RAID 卡因为官方也是软 RAID 这样可提供可视化报警。服务器改群晖需要注意群晖只支持部分硬件 RAID 卡，建议把阵列卡刷成 IT 直通模式这样兼容性会更好一些，请自行咨询服务器销售商确认 RAID 卡是否支持 IT 直通，群晖系统能否使用

# 开始教程

# 下载引导镜像

前往巴西大佬主页下载：https://github.com/fbelavenuto/arpl/releases

这个 arpl.img 就是引导镜像，先把这个下载下来

解压后会获得一个 arpl.img 引导镜像文件，需要使用工具把它写到我们准备好的 U 盘里

# U 盘刷写工具

推荐：rufus

前往下载：https://rufus.ie/downloads/

设备选择你的 U 盘，引导类型选择你刚刚解压出来的 arpl.img 引导镜像文件，点击开始等待写入完毕

# 开始引导

  1. 把刷写好的 U 盘接入要当 Nas 的机子，接上键盘，接上显示器，接上网线，开机

  2. 按 DEL（不同主板快捷键不一样）进入 BIOS 界面，设置 U 盘首选启动项（自行百度），按 F10 保存退出

  3. 重新进入时看到这个界面，它会自动进入

留意是否有滚动画面

正常显示出 ip 地址就可以在同局域网内的其它电脑浏览器输入这个地址就可以进入编译选单，如果 ip 都没显示说明网卡没有正常被驱动请检查好网卡。

接着选择 Build the loader，进行编译引导

在 params 参数处不输入任何内容，回车即可。再选择 Exit 返回上级菜单，继续选择 Build the loader，进行编译引导（不是十代处理器省略此步，此操作是为了添加核显驱动）

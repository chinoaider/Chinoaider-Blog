# 前言

没有公网 IP？

不是正版群晖？

没有公网 IP 或者不是正版群晖都没关系

你可以通过其他方式进行内网穿透

效果和 qc 一样而且操作简单

还能邀请任何用户加入你的尾网

废话不多说直接进入教程

参考文档：Docs · Tailscale Docs

# Taliscale

# 下载安装 Taliscale

首先在你的群晖套件中心下载一个叫 Taliscale 的工具，打开套件中心，搜索 Taliscale , 安装 Taliscale

注：如果没有搜索到的话，可以添加第三方套件源再进行下载安装，可以看这里：添加第三方套件源

安装后打开，会跳转到一个网页，提示让你登陆

选择你的登陆方式

我这里选择用 GitHub 授权登陆，你也可以使用其他方式登陆

点击连接

# 启用 Synology 出站连接

如果你运行的是 Synology DSM6 则可以跳过这一步，DSM6 会以 root 身份运行 Tailscale 。

Synology DSM7 对套件进行了更严格的限制，现在你可以在你的 Tailscale 控制台看到你的设备了，但是现在还用不了，需要 启用 Synology 出站连接

我用的是 Synology DSM7 ，所以需要进行以下配置，打开 控制面板 👉 任务计划 👉 新增 👉 触发的任务 👉 用户定义的脚本

常规 ，任务名称随意，用户账号改成 root ，事件选择开机

切换到 任务设置 菜单，复制粘贴以下命令到用户自定义的脚本，点击确定即可

选中刚刚创建的任务计划，右键运行或者按菜单按钮运行

会发现连接状态变成绿色，表示已经可以使用这个 IP 地址访问这台 Nas 了，实现内网穿透

勾选这个任务计划，启用这个任务计划，应用保存，下次开机就会自动运行这个脚本了

# 其他方式连接 Taliscale

你可以在电脑下载 Taliscale 工具外网访问群晖共享文件夹了 Download · Tailscale ，和上面的使用方式差不多，下载后安装，打开后在电脑托盘找到，登陆你的账号，连接即可，支持各种平台。

然后就可以用 RaiDrive 连接你的群晖用户，访问共享文件夹，这里改成你的主机 IP，然后用群晖的本地用户登陆即可，不过需要先在群晖套件中心下载 WebDav Server 配置允许使用 WebDav 方式访问，可以参考这篇：WebDav Server

手机也一样，在 Google play 商店搜索下载安装 Taliscale ，打开后同样先登陆，授权，连接，傻瓜式安装

使用时需要打开这个 Active，然后就可以在浏览器输入 IP 加 5000 端口，访问群晖

也可以用手机文件浏览器的 webdav 方式访问共享文件，推荐 es 文件浏览器（有这个功能的 app 都能使用），同样需要先在群晖套件中心下载 WebDav Server 配置允许使用 WebDav 方式访问

  • 新建 WebDav 服务器
  • 服务器：填你群晖外网 IP
  • 端口：5006
  • 加密：https
  • 用户名密码：你的群晖本地用户和密码
  • 重命名：随意

# 邀请用户加入您的尾网

在 Taliscale 的后台选择 user 栏，点击 Invite users 【邀请用户】，选择角色 Member【成员】，Generate & copy invite link【生成和复制邀请链接】，发送给你另外的设备按照步骤操作即可，详细看参考：邀请任何用户加入您的尾网・Tailscale 文档

有一点需要注意，每 6 个月需要重新授权一次，设备过期重新授权即可

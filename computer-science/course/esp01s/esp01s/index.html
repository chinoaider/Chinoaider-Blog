# 前言

使用 esp01sWiFi 模块远程控制、语音控制灯光，可接入小爱、小度，成本仅需 20 块左右

# 准备工作

# 所需硬件

CH340C 烧录器

esp01sWiFi 模块

WiFi 模块继电器

220v 转 5v 变压模块

220vLED 硬灯条

# 接线图

# 单火版

  • NO = Normal Open（常开触点）➜ 继电器线圈得电（吸合）后才导通，断电时断开

  • NC = Normal Closed（常闭触点）➜ 继电器线圈断电（释放）时就导通，吸合后断开

    开关接线：3v3 和 io2 接线到开关上（保留正常开关，无需求可不接）

    单火版需要电池给单片机供电，耗电大需要经常换电池

# 零火版

220v 转 5v 给单片机电路板供电

零火版可以用 AC220v 转 DC5v 给单片机电路板供电，实现永久供电

# 烧录程序

推荐使用” 巴法云 “，入手简单，配置方便

点灯科技只能接入一个 diy 设备

# 点灯科技

# 获取 key

  1. 在手机端下载点灯科技 app：blinker
  2. 在首页点击右上角”+“号添加设备
    • 添加【独立设备】
    • 点击【网络接入】
    • 注册设备后会出现【key】，后面会用到

# 安装 Arduino

  1. 前往官网下载安装 Arduino： https://www.arduino.cc/en/software/

    • 选择为所有用户安装

  2. 打开 Arduino 首选项里添加 ESP 开发板网址：http://arduino.esp8266.com/stable/package_esp8266com_index.json （如果是 Arduino ide 则不需要这一步）

  3. 工具安装库：

    • 搜索：blinker

    • 搜索：esp8266

  1. 开发板和端口配置
    • 端口选择 “COM3”，如果没有出现端口，则需要安装 "CH340C" 驱动，CH340C 烧录器自带驱动
    • 开发板选择：“Generic ESP8266 Module”

  1. 代码：

    • auth ：填写前面点灯科技 app 获得的 key

    • ssid ：WiFi 名称 (仅支持 2.4g)

    • pswd ：WiFi 密码

  1. 开始编译，先编译成功后再上传到 esp01s 模块上，

  2. 开始设计程序 UI

    • 打开点灯科技 app，这时会发现创建的独立设备会显示在线，点击进入
    • 点击右上角的 “🖊” 编辑界面
    • 添加一个 “按键”
    • 组件键名：btn-abc
    • 按键类型：开关按键
    • 显示文本：灯光控制（随意）
    • 设置完毕后点击右上角的 “🔒”

# 巴法云

# 获取私钥

  1. 前往官网注册巴法云： https://cloud.bemfa.com/
  2. 注册完左上角这个就是你的私钥，待会要用到先不管

  1. 添加设备：

    • 命名为： ESP01S002 (命名格式只支持英文和数字，具体看： https://cloud.bemfa.com/docs/src/speaker_mi.html)

    • 昵称：可以随便改（也可在 app 上改）

# 安装 Mixly

  1. 前往米思齐官网下载【Mixly】编程软件：米思齐官网： https://mixly.cn/

百度云盘： https://pan.baidu.com/s/1EdBmAlYFzJ_2rULhKm2g3g?pwd=ny1n

选择对应系统（Win10/11、macOS、Linux）的 “一键更新完整版” 即可

  1. 打开【Mixly】，选择对应的开发板进入界面，用的是【esp01s】模块所以这里选【esp8266】

  1. 开发板和端口配置

    • 端口选择 “COM3”，如果没有出现端口，则需要安装 "CH340C" 驱动，CH340C 烧录器自带驱动

    • 开发板选择：“WeMos D1 R1”

  1. 在界面中右上角的设置找到，管理库，在导入库里找到 【巴法】(bemfa)，打勾并点击云端导入

这时在侧边栏就能看到巴法云这一栏了，可以调用他的函数模块了，可以选择用模块 0 代码编程，也可以直接 c 语音编程

代码：将前面获取到的私钥填入 String UI="" 中

  1. 开始编译，编译失败的话查看软件路径有没有中文，或者用管理员权限打开，先编译成功，再上传到 esp01s 模块上

上传成功后，可以在巴法 app 端看到在线设备，现在就可以在 app 远程控制灯光了

# 接入小爱

# 配置步骤

  1. 创建设备主题：在巴法云控制台创建主题，确保主题名后缀符合设备类型规则
  2. 下载米家 app：下载安装并打开米家 app
  3. 添加设备：
    • 在” 我的 “页面下找到点击” 添加其他平台 “
    • 选择 "添加"
    • 搜索第三方平台 "点灯科技" 或” 巴法 “并选择
    • 输入巴第三方平台账号完成绑定
  4. 开始使用：设备将自动同步到米家，即可通过语音控制
  5. 更加详细步骤可查看：https://cloud.bemfa.com/docs/src/speaker_mi.html

# 接入小度

注意：想使用小度语音控制，至少需要一个小度音箱才能使用语音功能控制灯光

# 配置步骤

  1. 创建设备主题：在巴法云控制台创建主题，确保主题名后缀符合设备类型规则
  2. 下载小度 App：下载小度音箱 App 或小度 App
  3. 添加设备：
    • 在 App 首页点击 + 号
    • 选择 "添加设备"
    • 搜索第三方平台 "blinker" 或” 巴法 “并选择
    • 输入巴第三方平台账号完成绑定
  4. 开始使用：设备将自动同步到小度，即可通过语音控制
  5. 更加详细步骤可查看：https://cloud.bemfa.com/docs/src/speaker_du.html

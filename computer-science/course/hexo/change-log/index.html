# 卡在加载动画

发现时间：2026-03-07

一直卡在加载页面无法跳转进入网站

Uncaught SyntaxError: Identifier 't' has already been declared (at app.js?v=0.2.5:1:1)

# 报错原因：

shoka 主题的 pjax.js 里有一段 console.log 输出是作者的主题声明链接，在打开网站异步加载控件到最后一步时这个主题声明链接被浏览器当成广告拦截了，所以才一直卡在加载动画

# 解决方法：

找到项目路径下的.\hexo-blog\themes\shoka\source\js_app\pjax.js

找到这一段直接注释掉

或者把浏览器拦截广告之类的插件或设置关了
